# python基础：Python 运算符

Python 编程中，运算符和表达式是构建算术、逻辑和比较操作的关键。通过熟悉和灵活运用这些运算符和表达式，我们可以进行数值计算、条件判断和数据操作等各种任务。本文将深入探索 Python 中常用的运算符与表达式，并通过丰富的案例帮助您理解其用法和应用场景。

# 1. 算术运算符

算术运算符用于执行基本的数学运算。下面是 Python 中常用的算术运算符：

+：加法运算符，用于相加两个操作数。
-：减法运算符，用于从第一个操作数减去第二个操作数。
*：乘法运算符，用于相乘两个操作数。
/：除法运算符，用于将第一个操作数除以第二个操作数。
//：整除运算符，用于返回两个操作数的整数商。
%：取模运算符，用于返回两个操作数的余数。
**：幂运算符，用于将第一个操作数的值提升到第二个操作数的幂。

# 2.赋值运算符

赋值运算符用于将值分配给变量。除了简单的赋值运算符 =，Python 还提供了一些复合赋值运算符，可以将运算和赋值合并在一起。下面是 Python 中常用的赋值运算符：


=：简单的赋值运算符，用于将右侧的值分配给左侧的变量。
+=：加法赋值运算符，用于将右侧的值与左侧的变量相加，并将结果赋给左侧的变量。
-=：减法赋值运算符，用于将右侧的值从左侧的变量中减去，并将结果赋给左侧的变量。
*=：乘法赋值运算符，用于将右侧的值与左侧的变量相乘，并将结果赋给左侧的变量。
/=：除法赋值运算符，用于将左侧的变量除以右侧的值，并将结果赋给左侧的变量。
//=：整除赋值运算符，用于将左侧的变量整除右侧的值，并将结果赋给左侧的变量。
%=：取模赋值运算符，用于将左侧的变量对右侧的值取模，并将结果赋给左侧的变量。
**=：幂赋值运算符，用于将左侧的变量的值提升到右侧的幂，并将结果赋给左侧的变量。

# 3. 比较运算符

比较运算符用于比较两个值或表达式的关系，并返回布尔值（True 或 False）。下面是 Python 中常用的比较运算符：

# ==：等于运算符，用于检查两个操作数是否相等。
# !=：不等于运算符，用于检查两个操作数是否不相等。
# <：小于运算符，用于检查第一个操作数是否小于第二个操作数。
# >：大于运算符，用于检查第一个操作数是否大于第二个操作数。
# <=：小于等于运算符，用于检查第一个操作数是否小于或等于第二个操作数。
# >=：大于等于运算符，用于检查第一个操作数是否大于或等于第二个操作数。

# 4. 逻辑运算符

逻辑运算符用于对布尔值进行逻辑运算，并返回新的布尔值。下面是 Python 中常用的逻辑运算符：

and：与运算符，用于检查两个操作数是否都为 True。
or：或运算符，用于检查两个操作数是否至少有一个为 True。
not：非运算符，用于对操作数进行取反操作。

logical_or = a or b
logical_not_a = not a
logical_not_b = not b

print("Logical AND:", logical_and)
print("Logical OR:", logical_or)
print("Logical NOT (a):", logical_not_a)
print("Logical NOT (b):", logical_not_b)

# 5. 位运算符

位运算符用于对二进制数进行位级操作。虽然在日常编程中使用较少，但在某些特定情况下，位运算符可以提供高效的解决方案。下面是 Python 中常用的位运算符：


&：按位与运算符，用于对两个操作数的每个位执行与操作。
|：按位或运算符，用于对两个操作数的每个位执行或操作。
^：按位异或运算符，用于对两个操作数的每个位执行异或操作。
~：按位取反运算符，用于对操作数的每个位进行取反操作。
<<：左移运算符，用于将操作数的二进制表示向左移动指定的位数。
>>：右移运算符，用于将操作数的二进制表示向右移动指定的位数。

# 位运算符示例
a = 5  # 二进制表示为 0101
b = 3  # 二进制表示为 0011

bitwise_and = a & b     # 0001，结果为 1
bitwise_or = a | b      # 0111，结果为 7
bitwise_xor = a ^ b     # 0110，结果为 6
bitwise_not_a = ~a      # 1010，结果为 -6（按补码表示）
bitwise_left_shift = a << 2   # 010100，结果为 20
bitwise_right_shift = a >> 1  # 0010，结果为 2

print("Bitwise AND:", bitwise_and)
print("Bitwise OR:", bitwise_or)
print("Bitwise XOR:", bitwise_xor)
print("Bitwise NOT (a):", bitwise_not_a)
print("Bitwise Left Shift:", bitwise_left_shift)
print("Bitwise Right Shift:", bitwise_right_shift)

# 6.成员运算符

成员运算符用于检查一个值是否属于某个序列（如字符串、列表或元组）或集合中。Python 提供了两个成员运算符：

in：如果指定的值在序列中存在，则返回 True；否则返回 False。
not in：如果指定的值在序列中不存在，则返回 True；否则返回 False。
1
2

下面是成员运算符在 Python 中的使用示例：

# 成员运算符示例
fruits = ['apple', 'banana', 'cherry']

print('apple' in fruits)     # True
print('orange' in fruits)    # False
print('orange' not in fruits) # True

# 7.身份运算符

身份运算符用于比较对象的内存地址，判断两个变量是否引用同一个对象。Python 提供了两个身份运算符：

is：如果两个变量引用同一个对象，则返回 True；否则返回 False。
is not：如果两个变量引用的不是同一个对象，则返回 True；否则返回 False。

以下是身份运算符在 Python 中的使用示例：

# 身份运算符示例
x = 5
y = 5
z = [1, 2, 3]
w = [1, 2, 3]

print(x is y)    # True
print(z is w)    # False
print(z is not w) # True

# 8.python运算符的优先级

运算符优先级是指在表达式中，当有多个运算符出现时，先执行哪个运算符的规则。在Python中，运算符的优先级由高到低依次是括号、指数运算、正负号和位取反运算、乘除法和取余、加法和减法、位移运算、比较运算、身份运算、成员运算和逻辑运算。优先级高的运算符会先被执行，如果优先级相同，则从左到右依次执行。

运算符优先级的存在是为了解决表达式中可能存在多种运算时，如何安排运算顺序的问题。如果不存在优先级，必须通过括号来明确运算顺序，这会导致代码难以阅读和维护。有了运算符优先级的规定，程序员可以更加灵活和高效地编写代码，减少出错的可能性。

在实际编程中，如果对于优先级不是很确定，可以使用括号来明确指定运算的顺序，从而避免潜在的错误和混淆。同时，也尽量避免写出过于复杂的表达式，以提高代码的可读性和可维护性。 Python运算符的优先级从高到低如下：

1、括号 ()：最高优先级，先执行括号里的运算。
2、指数运算 **：执行指数运算。
3、正负号 +x, -x, ~x：执行正负号和位取反运算。
4、乘法、除法和取余 *, /, //, %：执行乘法、除法和取余运算。
5、加法和减法 +, -：执行加法和减法运算。
6、位移 <<, >>：执行位移运算。
7、比较运算符 <=, <, >, >=, !=, ==：执行比较运算。
8、身份运算符 is, is not：执行身份运算。
9、成员运算符 in, not in：执行成员运算。
10、逻辑运算符 not 或 or 和 and：执行逻辑运算。
