函数

在 Python 中，函数是一段可重用的代码块，用于执行特定的任务。

定义函数：

使用 def 关键字来定义函数，语法如下：

def function_name(parameters):
    # 函数体
    return result  # 可选的返回值
1
2
3

参数：

参数分为位置参数、默认参数、关键字参数和可变参数。

  • 位置参数：按照参数定义的顺序传递实参。
  • 默认参数：在定义函数时为参数指定默认值，如果调用时未提供该参数的值，则使用默认值。
  • 关键字参数：通过参数名来传递值。
  • 可变参数：包括 *args（接收任意数量的位置参数）和 **kwargs（接收任意数量的关键字参数）。

返回值：

使用 return 语句返回函数的结果。如果没有 return 语句，函数默认返回 None。

函数的优点：

  1. 提高代码的复用性，避免重复编写相同的代码逻辑。
  2. 使代码结构更清晰，易于理解和维护。

示例：

# 定义一个带有位置参数的函数
def add_numbers(a, b):
    return a + b

# 调用函数
result = add_numbers(3, 5)
print(result)  # 输出: 8

# 定义带有默认参数的函数
def greeting(name="World"):
    return f"Hello, {name}!"

print(greeting())  # 输出: Hello, World!
print(greeting("Alice"))  # 输出: Hello, Alice!

# 定义带有可变参数的函数
def print_info(*args, **kwargs):
    for arg in args:
        print(arg)
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(1, 2, 3, name="John", age=25)

# 函数的常见用法

  1. 函数的嵌套 可以在一个函数内部定义另一个函数。内部函数只能在外部函数内部被调用，外部无法直接访问。

    def outer_function():
    def inner_function():
        print("This is the inner function")
    inner_function()

def fibonacci(n):
    fib_seq = [0, 1]
    while len(fib_seq) < n:
        fib_seq.append(fib_seq[-1] + fib_seq[-2])
    return fib_seq

  1. 对字符串进行加密（简单的恺撒加密）
def caesar_encrypt(text, shift):
    encrypted_text = ""
    for char in text:
        if char.isalpha():
            if char.islower():
                encrypted_text += chr((ord(char) - ord('a') + shift) % 26 + ord('a'))
            else:
                encrypted_text += chr((ord(char) - ord('A') + shift) % 26 + ord('A'))
        else:
            encrypted_text += char
    return encrypted_text

print(caesar_encrypt("Hello", 3))

def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        return n * factorial(n - 1)

  1. 合并两个已排序的列表为一个新的已排序列表
def merge_sorted_lists(list1, list2):
    merged_list = []
    i = j = 0
    while i < len(list1) and j < len(list2):
        if list1[i] < list2[j]:
            merged_list.append(list1[i])
            i += 1
        else:
            merged_list.append(list2[j])
            j += 1
    merged_list.extend(list1[i:])
    merged_list.extend(list2[j:])
    return merged_list

list1 = [1, 3, 5, 7]
list2 = [2, 4, 6, 8]
print(merge_sorted_lists(list1, list2))
