# python 数据类型

Python 是一种动态类型语言，拥有丰富的内置数据类型。了解这些数据类型是编写高效、可靠的 Python 程序的关键。本文将详细介绍 Python 中的所有数据类型，并通过案例进行详细解读，帮助你深入理解它们的特性和用法。

# 1 数字类型（Number）

Python 提供了多种数字类型，包括整数（int）、浮点数（float）、复数（complex）等。数字类型用于存储和操作数值数据。 案例：计算圆的面积 让我们以计算圆的面积为例来演示数字类型的使用：

import math

radius = 5
area = math.pi * radius ** 2

print("圆的面积为:", area)

# 2 字符串类型（String）

字符串是由字符组成的序列，用于表示文本数据。Python 中的字符串是不可变的，可以使用单引号或双引号括起来。

案例：字符串拼接 下面的案例演示了字符串拼接的用法：

first_name = "John"
last_name = "Doe"
full_name = first_name + " " + last_name

print("完整姓名:", full_name)

# 3 列表类型（List）

列表是一种有序、可变的数据类型，用于存储多个元素。列表使用方括号 [ ] 表示，每个元素之间用逗号分隔。

案例：购物清单 下面的案例展示了如何使用列表来表示购物清单：

shopping_list = ["苹果", "香蕉", "橙子", "西瓜"]

print("购物清单:")
for item in shopping_list:
    print(item)

# 4 元组类型（Tuple）

元组是一种有序、不可变的数据类型，类似于列表。元组使用圆括号 ( ) 表示，每个元素之间用逗号分隔。

案例：学生信息 下面的案例展示了如何使用元组来表示学生的信息：

student = ("John", 20, "Male")

print("学生信息:")
print("姓名:", student[0])
print("年龄:", student[1])
print("性别:", student[2])

# 5 字典类型（Dictionary）

字典是一种无序、可变的数据类型，用于存储键值对。字典使用花括号 { } 表示，每个键值对之间用冒号 : 分隔。

案例：学生成绩 下面的案例展示了如何使用字典来表示学生成绩： scores = {"数学": 90, "英语": 85, "科学": 抱歉，似乎我在案例的最后截断了。以下是继续的部分：

scores = {"数学": 90, "英语": 85, "科学": 92}

print("学生成绩:")
for subject, score in scores.items():
    print(subject + ":", score)

# 6 集合类型（Set）

集合是一种无序、不重复的数据类型，用于存储唯一的元素。集合使用花括号 { } 表示，每个元素之间用逗号分隔。

案例：去重 下面的案例展示了如何使用集合来去除列表中的重复元素

numbers = [1, 2, 3, 3, 4, 4, 5]
unique_numbers = set(numbers)

print("去重后的列表:")
for number in unique_numbers:
    print(number)

# 总结

本文深入介绍了 Python 中的所有数据类型，包括数字类型、字符串类型、列表类型、元组类型、字典类型和集合类型。通过案例的详细解读，你应该对这些数据类型的特性和用法有了更深入的理解。掌握这些数据类型将为你编写高效、可靠的 Python 程序提供基础。
