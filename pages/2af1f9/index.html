# Python基础：掌握while循环语句

在Python编程中，循环语句是一种重要的控制结构，它允许我们重复执行一段代码块。其中一种常见的循环类型是while循环。本文将介绍while循环的基本语法和使用方法，并给出多个实例案例，帮助你更好地理解和应用while循环。

# 1 简介

while循环是一种条件控制结构，它在给定条件为真（True）的情况下重复执行一段代码块。它的基本语法如下：

while 条件:
    # 代码块
1
2

在每次循环迭代中，首先会检查条件是否为真。如果条件为真，则执行代码块；如果条件为假，则跳过代码块，继续执行程序的下一部分。

# 2 while循环案例

# 案例1：计数器

假设我们要编写一个程序，打印出数字从1到5的计数器。可以使用while循环来实现：

# 代码示例：
counter = 1
while counter <= 5:
    print(counter)
    counter += 1
1
2
3
4
5

上述代码中，我们首先初始化计数器变量counter为1。然后，while循环会检查counter <= 5的条件是否为真。如果为真，则打印出当前计数器的值，并将计数器加1。这个过程会一直重复，直到计数器的值大于5，循环才会结束。

# 案例2：猜数字游戏

需求：让我们编写一个简单的猜数字游戏。程序随机生成一个1到10之间的整数，然后要求用户猜这个数字是多少，直到猜中为止。

import random

number = random.randint(1, 10)
guess = 0

while guess != number:
    guess = int(input("猜一个1到10之间的数字："))

print("恭喜你，猜对了！")

在上述代码中，我们使用random模块生成一个随机整数，并将其赋值给变量number。然后，我们使用while循环来提示用户输入猜测的数字，并将其赋值给变量guess。循环会一直执行，直到guess的值等于number，即猜中了数字。

案例3：密码验证 需求：使用while循环来实现一个简单的密码验证程序。要求用户输入密码，如果输入的密码不正确，则继续提示输入，直到输入正确为止

password = "password"
input_password = ""

while input_password != password:
    input_password = input("请输入密码：")

print("密码正确，登录成功！")

总结： while循环允许我们根据条件重复执行代码块，直到条件不满足为止。我们通过计数器、猜数字游戏和密码验证等案例演示了while循环的实际应用。掌握while循环将为你在编写Python程序时提供更大的灵活性和控制能力
